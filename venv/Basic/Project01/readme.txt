프로젝트 01 로또 시뮬레이션
규칙
로또는 주 1회씩 열림.
하지만 한 사람이 한 회차에 여러 번 참여 가능
번호 1 ~ 45
주최측에서는 매주 6개의 ‘일반 당첨 번호’ 와 1개의 ‘보너스 번호’ 추첨
참가자는 1번 참여할 때마다 서로 다른 번호 6개를 선택

당첨 액수는 아래 규칙
내가 뽑은 번호 6개와 일반 당첨 번호 6개 모두 일치 (10억)
내가 뽑은 번호 5개와 일반 당첨 번호 5개 일치, 그리고 내 번호 1개와 보너스 번호 일치 (5천만)
번호 5개와 일반 5개 (100만)
4개와 4개 (5만)
3개 3개 (5천)

필요 함수
generate_numbers
파라미터 정수 n
무작위로 1 ~ 45 사이의 서로 다른 번호 n개를 뽑고, 그 번호들이 담긴 리스트 리턴
참고로 이 함수는 참가자의 번호 뽑는데와 보너스를 포함한 당첨 번호 7개를 뽑는데 쓰임

draw_winning_numbers
일반 당첨 번호 6개와 보너스 번호 1개가 포함된 리스트 리턴
일반 당첨 번호 6개는 정렬되어 있어야 하고, 보너스 번호는 마지막에 추가
앞서 정의한 generate_numbers 함수 활용

count_matching_numbers
파리미터로 리스트 list_1과 리스트 list_2를 받고, 두 리스트 사이에 겹치는 번호 개수 리턴

check
참가자의 당첨 금액 리턴
파라미터로 참가자가 뽑은 번호가 담긴 리스트 numbers와 주최측에서 뽑은 번호가 담긴 리스트 winning_numbers 받음
numbers는 번호 6개 winning_numbers 7개
